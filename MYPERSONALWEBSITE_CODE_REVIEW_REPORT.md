**影响**: 代码冗余，构建警告  
**修复**: 已通过 ESLint 自动修复

#### 2. 性能优化建议
**问题**: 6 个 chunks 超过 1000KB  
**建议**: 
```typescript
// vite.config.ts 中的优化配置
manualChunks(id) {
  if (id.includes('node_modules')) {
    // 进一步细分 vendor 包
    if (id.includes('vue')) return 'vue-vendor'
    if (id.includes('gsap')) return 'animation-vendor'
    if (id.includes('lucide')) return 'ui-vendor'
    return 'vendor'
  }
}
```
