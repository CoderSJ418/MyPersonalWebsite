## 一、项目信息

### 1.1 基本信息

| 属性 | 值 |
|------|-----|

| 技术栈 | Vue 3.4.15 + TypeScript 5.3.3 + Vite 5.0.12 + Pinia 2.1.7 |

### 1.2 项目目标

**主要目标**:
- 打造一个集「技术展示、经验分享、风格定制」于一体的个人品牌平台
- 重点落地 AI 提示词工程、UI/UX 设计、前端工程化三大能力
- 为前端开发者提供高质量的实践参考

**成功标准**:
- Lighthouse 评分 ≥90 分
- 首屏加载时间 ≤2s
- 核心功能完整可用
- 单元测试覆盖率 ≥80%

### 1.3 项目范围

**包含范围**:
- 博客模块（文章列表、文章详情、分类检索、搜索功能）
- 作品集模块（作品展示、作品详情、筛选排序）
- 个人介绍页面
- 调色板功能（配色方案、UI 风格、字体组合）
- 性能优化（代码分割、图片优化、虚拟列表、缓存）
- 测试体系（单元测试、集成测试、E2E 测试）

**不包含范围**:
- AI 内容生成（迭代功能）
- 暗黑模式（迭代功能）
- 阅读统计（迭代功能）
- 用户系统（无用户登录）
- 数据库（静态数据 + localStorage）
