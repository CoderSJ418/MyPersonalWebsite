# 生产力工具

通用工作流工具，不针对代码。

- **[caveman](./caveman/SKILL.md)** — 极度压缩通信模式。通过删除填充词来节省 ~75% token，同时保持完整的技术准确性。
- **[grill-me](./grill-me/SKILL.md)** — 被不遗余力地采访关于计划或设计，直到决策树的每个分支都得到解决。
- **[handoff](./handoff/SKILL.md)** — 将当前对话压缩成交接文档，以便另一个 agent 继续工作。
- **[teach](./teach/SKILL.md)** — 在多个会话中教用户新技能或概念，使用当前目录作为有状态的教学工作区。
- **[write-a-skill](./write-a-skill/SKILL.md)** — 创建具有适当结构、递进式展示和捆绑资源的新 skills。
