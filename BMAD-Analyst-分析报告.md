## 🎯 项目概述

### 技术栈
- **前端框架**: Vue 3.4.15 (Composition API)
- **构建工具**: Vite 5.0.12
- **状态管理**: Pinia 2.1.7
- **路由**: Vue Router 4.2.5
- **样式框架**: Tailwind CSS 3.4.1
- **动画库**: GSAP 3.14.2
- **图标库**: Lucide Vue Next 0.312.0
- **TypeScript**: 5.3.3
- **测试框架**: Vitest + Playwright
