## 🚀 立即开始

**Windows用户**：
```bash

```

```bash
# 进入项目目录
cd E:\work\AI\MyPersonalWebsite

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

服务器将在 `http://localhost:5173` 启动。

## 📦 项目特性

### ✨ 核心功能
- 🏠 **首页**: 个人介绍、技术栈、精选项目
- 💼 **项目展示**: 项目列表、项目详情、项目筛选
- 🎯 **技能展示**: 技能分类、技能熟练度
- 📝 **技术博客**: 文章列表、文章详情、文章标签
- 📧 **联系方式**: 联系表单、社交媒体

### 🎨 设计亮点
- 🌙 **暗黑模式**: 主题切换支持
- 📱 **响应式设计**: 移动端优先
- ✨ **流畅动画**: 过渡和微交互
- ♿ **可访问性**: 键盘导航、屏幕阅读器

### 🛠️ 技术栈
- **框架**: Vue 3.4.x (Composition API)
- **语言**: TypeScript 5.3.x
- **构建**: Vite 5.0.x
- **状态**: Pinia 2.1.x
- **路由**: Vue Router 4.2.x
- **样式**: Tailwind CSS 3.4.x

## 🎯 下一步

### 1. 启动项目
```bash
# Windows用户
START.bat

# 或使用命令行
npm run dev
```

### 2. 浏览网站
打开浏览器访问 `http://localhost:5173`

### 3. 自定义内容
编辑以下文件来个性化您的网站：
- `src/assets/data/projects.json` - 项目数据
- `src/assets/data/skills.json` - 技能数据
- `src/assets/data/blog-index.json` - 博客索引
- `index.html` - 网站标题和描述

### 4. 构建生产版本
```bash
# Windows用户
BUILD.bat

# 或使用命令行
npm run build
```

### 5. 部署上线
将 `dist` 目录部署到：
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

## 💡 提示

1. **首次运行**: 首次运行需要安装依赖，可能需要几分钟
2. **端口占用**: 如果5173端口被占用，Vite会自动使用其他端口
3. **热更新**: 开发模式下修改代码会自动刷新浏览器
4. **暗黑模式**: 点击页面右上角的月亮/太阳图标切换主题
5. **响应式**: 调整浏览器窗口大小查看响应式效果
