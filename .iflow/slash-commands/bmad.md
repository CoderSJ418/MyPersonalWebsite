# BMAD 工作流命令

**description**: BMAD 方法论工作流，包含多个专业代理

## 可用命令

### 分析师代理
```
/bmad/agents/bmad-analyst - 业务分析师，负责需求分析、问题诊断和解决方案制定
```

### 架构师代理
```
/bmad/agents/bmad-architect - 架构师，负责系统架构设计和技术选型
```

### 开发者代理
```
/bmad/agents/bmad-dev - 开发者，负责代码实现和功能开发
```

### UX 专家代理
```
/bmad/agents/bmad-ux-expert - UX 专家，负责用户体验和界面设计
```

### 产品经理代理
```
/bmad/agents/bmad-pm - 产品经理，负责产品规划和需求管理
```

### 技术架构师代理
```
/bmad/agents/bmad-tea - 技术架构师，负责技术架构设计
```

## 使用示例
```
/bmad/agents/bmad-analyst - 帮我分析一下这个需求
/bmad/agents/bmad-architect - 设计一个微服务架构
/bmad/agents/bmad-dev - 实现这个功能
```
