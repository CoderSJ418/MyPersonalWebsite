# 佘杰的个人网站

7年前端开发工程师，Vue专家，专注于前端技术栈和工程化。

## 技术栈

- **框架**: Vue 3.4.x (Composition API)
- **语言**: TypeScript 5.3.x
- **构建工具**: Vite 5.0.x
- **状态管理**: Pinia 2.1.x
- **路由**: Vue Router 4.2.x
- **样式**: Tailwind CSS 3.4.x
- **图标**: Lucide Vue Next

## 功能特性

- 首页：个人介绍、技术栈、精选项目
- 项目展示：项目列表、项目详情
- 技能展示：技能分类、技能熟练度
- 技术博客：文章列表、文章详情
- 联系方式：联系表单、社交媒体
- 暗黑模式：主题切换
- 响应式设计：移动端优先

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发

```bash
npm run dev
```

### 构建

```bash
npm run build
```

### 预览

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

### 代码格式化

```bash
npm run format
```

## 项目结构

```
MyPersonalWebsite/
├── public/                    # 静态资源
├── src/
│   ├── assets/               # 资源文件
│   │   ├── data/            # 本地数据
│   │   └── styles/          # 全局样式
│   ├── components/          # 组件
│   │   ├── common/          # 通用组件
│   │   ├── home/            # 首页组件
│   │   ├── projects/        # 项目组件
│   │   ├── skills/          # 技能组件
│   │   ├── blog/            # 博客组件
│   │   └── contact/         # 联系组件
│   ├── composables/         # 组合式函数
│   ├── stores/              # 状态管理
│   ├── router/              # 路由
│   ├── types/               # 类型定义
│   ├── utils/               # 工具函数
│   ├── views/               # 页面
│   ├── App.vue              # 根组件
│   └── main.ts              # 入口文件
├── docs/                    # 文档
├── index.html
├── package.json
├── tsconfig.json
├── vite.config.ts
└── tailwind.config.js
```
